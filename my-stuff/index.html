Vuiiger

A Game Boy "Mystery Dungeon" engine, making extensive use of the console's limited video capabilities. Features 8 independant entities which each can have a unique color palette and set of graphics, procedurally generated levels, a powerful variable-width-font engine which makes text compact and easy to read, and a custom scripting language created from scratch for this project, evscript.

evscript

A simple yet versatile programming language originally made for the Game Boy. Intended to replace the macro based scripting languages often used in assembly projects, evscript provides control flow structures, variable declarations and operators, and compact bytecode output which saves space compared to C or even some assembly code.
